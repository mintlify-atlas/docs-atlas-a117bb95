Novel is an open-source Notion-style WYSIWYG editor with AI-powered autocompletions. It provides a rich text editing experience with slash commands, bubble menus, and intelligent AI completions to help you write faster.

## Key features

Novel combines the best of modern text editing with powerful AI capabilities:

- **AI-powered autocompletions** - Type `++` to trigger intelligent text generation based on your context
- **Slash commands** - Access formatting options and tools with `/` command palette
- **Bubble menu** - Contextual formatting toolbar appears when you select text
- **Image uploads** - Drag & drop or paste images directly into the editor
- **Rich formatting** - Support for headings, lists, code blocks, links, and more
- **Task lists** - Create interactive checklists with checkbox support
- **Extensible** - Built on Tiptap with support for custom extensions
- **Local storage** - Automatic content persistence with configurable storage keys

## Technology stack

Novel is built on a modern tech stack:

- **Tiptap** - Headless editor framework for rich text editing
- **React** - Component-based UI library
- **Vercel AI SDK** - Streaming AI completions
- **OpenAI** - GPT-powered text generation
- **TailwindCSS** - Utility-first styling

## Cross-framework support

While Novel is built for React, community-maintained packages are available for other frameworks:

- [Svelte](https://novel.sh/svelte)
- [Vue](https://novel.sh/vue)
