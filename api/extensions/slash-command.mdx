## Overview

The `SlashCommand` extension provides a command palette that appears when users type `/` in the editor. It offers quick access to formatting options, AI features, and custom commands.

## Configuration

The SlashCommand extension is pre-configured with default items and render behavior. It's included in the default Novel extensions.

```typescript
const SlashCommand = Command.configure({
  suggestion: {
    items: getSuggestionItems,
    render: renderItems,
  },
});
```

## Usage Examples

### Basic Usage

The SlashCommand extension is included by default, so you don't need to add it manually:

```tsx
import Editor from "novel";

export default function MyEditor() {
  // SlashCommand is already included
  return <Editor />;
}
```

### Using Slash Commands

1. Type `/` anywhere in the editor
2. A menu will appear with available commands
3. Type to filter commands (e.g., `/head` shows heading options)
4. Use arrow keys to navigate
5. Press Enter to execute the selected command

### Keyboard Navigation

- **ArrowUp**: Move to previous command
- **ArrowDown**: Move to next command  
- **Enter**: Execute selected command
- **Escape**: Close the menu

## Customization

### Adding Custom Commands

To add custom slash commands, you'll need to create your own version of the SlashCommand extension:

```typescript
import { Extension } from "@tiptap/core";
import Suggestion from "@tiptap/suggestion";

const CustomSlashCommand = Extension.create({
  name: "slash-command",
  addOptions() {
    return {
      suggestion: {
        char: "/",
        command: ({ editor, range, props }) => {
          props.command({ editor, range });
        },
      },
    };
  },
  addProseMirrorPlugins() {
    return [
      Suggestion({
        editor: this.editor,
        ...this.options.suggestion,
      }),
    ];
  },
});
```

### Custom Command Items

```typescript
const customCommands = [
  {
    title: "Custom Command",
    description: "Does something custom",
    icon: <YourIcon />,
    command: ({ editor, range }) => {
      editor.chain().focus().deleteRange(range).run();
      // Your custom logic here
    },
  },
];
```

<Warning>
  Creating custom slash commands requires replacing the default SlashCommand extension. Make sure to remove the default extension from your configuration if you're adding a custom one.
</Warning>

## Implementation Details

- Built on top of [Tiptap Suggestion](https://tiptap.dev/docs/editor/api/utilities/suggestion)
- Uses [Tippy.js](https://atomiks.github.io/tippyjs/) for positioning
- Integrates with the AI completion API for "Continue writing" command
- Filters commands based on title, description, and search terms
- Automatically scrolls to keep selected item visible
