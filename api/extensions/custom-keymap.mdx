## Overview

The `CustomKeymap` extension enhances the default keyboard behavior in the editor. It provides improved text selection with Cmd/Ctrl+A, allowing users to first select text within a node before selecting the entire document.

## Source Location

Defined in `~/workspace/source/packages/core/src/ui/editor/extensions/custom-keymap.ts:15-56`

## Features

- Enhanced Cmd/Ctrl+A behavior
- First press: Select text within current node
- Second press: Select entire document
- Maintains standard selection behavior

## Configuration

The CustomKeymap extension is pre-configured and included in the default Novel extensions.

```typescript

  name: "CustomKeymap",
  addCommands() { /* ... */ },
  addKeyboardShortcuts() { /* ... */ },
});
```

### Keyboard Shortcuts

**Cmd/Ctrl + A** (Select All):
1. First press: Selects all text within the current node (paragraph, heading, etc.)
2. Second press: Selects the entire document

This two-step selection matches the behavior in many modern editors like Notion and Google Docs.

## Implementation Details

### Selection Logic

The extension determines selection behavior by comparing:
- Current selection start/end positions
- Current node start/end positions

If the selection doesn't extend to node boundaries, it expands to fill the node. Otherwise, it returns `false` to allow default behavior (select all).

```typescript
const startSelectionPos = tr.selection.from;
const endSelectionPos = tr.selection.to;
const startNodePos = tr.selection.$from.start();
const endNodePos = tr.selection.$to.end();

const isCurrentTextSelectionNotExtendedToNodeBoundaries =
  startSelectionPos > startNodePos || endSelectionPos < endNodePos;

if (isCurrentTextSelectionNotExtendedToNodeBoundaries) {
  editor.chain().selectTextWithinNodeBoundaries().run();
  return true;
}
return false;
```

## Usage in Custom Extensions

You can use the `selectTextWithinNodeBoundaries` command in your own extensions:

```typescript
import { Extension } from "@tiptap/core";

const MyExtension = Extension.create({
  name: "myExtension",
  
  addKeyboardShortcuts() {
    return {
      "Mod-Shift-a": ({ editor }) => {
        // First select within node
        editor.chain().selectTextWithinNodeBoundaries().run();
        // Then do something else
        return true;
      },
    };
  },
});
```

- `selectAll`: Select entire document
- `selectParentNode`: Select parent node
