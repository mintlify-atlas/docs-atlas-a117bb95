Novel uses Tailwind CSS with a custom design system. This guide shows you how to style the editor and its components.

## CSS Architecture

Novel includes three main CSS files:

1. `index.css` - CSS variables and theme colors
2. `tailwind.css` - Tailwind directives
3. `prosemirror.css` - ProseMirror-specific styles

All classes are prefixed with `novel-` to avoid conflicts with your application's styles.

## Customizing Colors

You can override the default colors by redefining the CSS variables:

```css
/* app/globals.css */
:root {
  /* Override with your brand colors */
  --novel-stone-400: rgb(100 116 139); /* slate-500 */
  --novel-stone-600: rgb(71 85 105); /* slate-600 */
  
  /* Custom highlight colors */
  --novel-highlight-blue: #dbeafe; /* blue-100 */
  --novel-highlight-green: #dcfce7; /* green-100 */
}
```

## ProseMirror Styles

Novel includes comprehensive ProseMirror styles. Here are the key classes you can customize:

```css
/* Editor container padding */
.ProseMirror {
  @apply novel-p-12 novel-px-8 sm:novel-px-12;
}

/* Placeholder text */
.ProseMirror .is-editor-empty:first-child::before {
  content: attr(data-placeholder);
  float: left;
  color: var(--novel-stone-400);
  pointer-events: none;
  height: 0;

/* Image styles */
.ProseMirror img {
  transition: filter 0.1s ease-in-out;

  &:hover {
    cursor: pointer;
    filter: brightness(90%);
  }

  &.ProseMirror-selectednode {
    outline: 3px solid #5abbf7;
    filter: brightness(90%);
  }
}

/* Image upload placeholder */
.img-placeholder {
  position: relative;

  &:before {
    content: "";
    box-sizing: border-box;
    position: absolute;
    top: 50%;
    left: 50%;
    width: 36px;
    height: 36px;
    border-radius: 50%;
    border: 3px solid var(--novel-stone-200);
    border-top-color: var(--novel-stone-800);

  }

@keyframes spinning {
  to {
    transform: rotate(360deg);
  }
}
```

## Styling Text Elements

Novel uses Tiptap's StarterKit with custom HTML attributes:

```tsx
// Bullet lists
bulletList: {
  HTMLAttributes: {
    class: "novel-list-disc novel-list-outside novel-leading-3 novel--mt-2",
  },
},

// Ordered lists
orderedList: {
  HTMLAttributes: {
    class: "novel-list-decimal novel-list-outside novel-leading-3 novel--mt-2",
  },
},

// List items
listItem: {
  HTMLAttributes: {
    class: "novel-leading-normal novel--mb-2",
  },
},

// Blockquotes
blockquote: {
  HTMLAttributes: {
    class: "novel-border-l-4 novel-border-stone-700",
  },
},

// Code blocks
codeBlock: {
  HTMLAttributes: {
    class: "novel-rounded-sm novel-bg-stone-100 novel-p-5 novel-font-mono novel-font-medium novel-text-stone-800",
  },
},

// Inline code
code: {
  HTMLAttributes: {
    class: "novel-rounded-md novel-bg-stone-200 novel-px-1.5 novel-py-1 novel-font-mono novel-font-medium novel-text-stone-900",
    spellcheck: "false",
  },
},
```

To override these styles, create more specific CSS selectors:

```css
.my-editor .ProseMirror blockquote {
  @apply border-l-4 border-blue-500 pl-4 italic text-gray-700;
}

.my-editor .ProseMirror code {
  @apply bg-purple-100 text-purple-800 px-2 py-1 rounded;
}

.my-editor .ProseMirror pre code {
  @apply bg-gray-900 text-gray-100 block p-4 rounded-lg;
}
```

## Custom Fonts

Novel uses the Cal Sans font for headings. You can customize this:

<Steps>

### Step 1: Define your font

```css
@font-face {
  font-family: "Your Custom Font";
  src: url("/fonts/your-font.woff2") format("woff2");
  font-weight: 600;
  font-display: swap;
}
```

### Step 2: Update CSS variable

```css
:root {
  --font-title: "Your Custom Font", sans-serif;
}
```

### Step 3: Override editor attributes

```tsx
import { Editor } from "novel";
import { EditorProps } from "@tiptap/pm/view";

const customEditorProps: EditorProps = {
  attributes: {
    class: "prose prose-lg font-serif focus:outline-none",
  },
};

<Editor editorProps={customEditorProps} />;
```

</Steps>

## Responsive Styling

Novel includes responsive styles. Here's how to customize them:

```css
/* Mobile */
@media screen and (max-width: 768px) {
  .ProseMirror {
    @apply px-4 py-8;
  }

  .drag-handle {
    display: none;
    pointer-events: none;
  }

  ul[data-type="taskList"] li > label {
    margin-right: 0.5rem;
  }
}

/* Desktop */
@media screen and (min-width: 768px) {
  .ProseMirror {
    @apply px-12 py-12;
  }
}
```

## Complete Custom Theme Example

Here's a complete example of a custom blue theme:

```css
/* app/editor-theme.css */
:root {
  /* Override color palette */
  --novel-stone-50: rgb(239 246 255); /* blue-50 */
  --novel-stone-100: rgb(219 234 254); /* blue-100 */
  --novel-stone-200: rgb(191 219 254); /* blue-200 */
  --novel-stone-400: rgb(96 165 250); /* blue-400 */
  --novel-stone-600: rgb(37 99 235); /* blue-600 */
  --novel-stone-800: rgb(30 64 175); /* blue-800 */
  
  /* Custom highlights */
  --novel-highlight-blue: rgb(191 219 254);
  --novel-highlight-default: rgb(255 255 255);
}

/* Custom editor container */
.blue-editor {
  @apply rounded-2xl border-2 border-blue-200 bg-gradient-to-br from-blue-50 to-white shadow-2xl;
}

.blue-editor .ProseMirror {
  @apply text-blue-900;
}

.blue-editor .ProseMirror blockquote {
  @apply border-blue-500 bg-blue-50 pl-4 py-2;
}

.blue-editor .ProseMirror code {
  @apply bg-blue-100 text-blue-800;
}

.blue-editor .ProseMirror a {
  @apply text-blue-600 hover:text-blue-800;
}
```

```tsx
import { Editor } from "novel";
import "./editor-theme.css";

export default function BlueEditor() {
  return <Editor className="blue-editor" />;
}
```
